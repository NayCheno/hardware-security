
\section{总览与使用说明}
\begin{frame}{Report-slide 目录目标}
\small
\begin{itemize}
  \item 将现有 \texttt{domain.md} 的小方向三篇精选材料，从单一矩阵拆成可编译、可维护、可扩展的 LaTeX slide 目录。
  \item 每个小方向一个子目录，包含 \texttt{section.tex} 与 \texttt{papers.yml}；后续可继续补图、表、实验细节和讲者备注。
  \item 版式参考上传 PDF 的结构：先给方向背景与技术演进，再对每篇文章按“内容摘要、研究背景、解决方案、实验结果、文章评价”展开。
\end{itemize}
\end{frame}
