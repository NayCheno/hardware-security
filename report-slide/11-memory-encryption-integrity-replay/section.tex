{\large\bfseries\color{Ink} 本地 AMD PDF 的机制是 VMEXIT/VMRUN 时由硬件保存和恢复加密 register state，并用受保护 integrity-check value 检查篡改。\par}\vspace{1.8mm}
\begin{columns}[T,totalwidth=\textwidth]

      \PanelTitle{讲解逻辑}
      \scriptsize \begin{itemize}
  \item VMRUN 进入 guest 时，硬件加载 guest register information。
  \item VMEXIT 时，硬件自动保存 guest register state。
  \item 保存区域使用 VM memory encryption key 加密，hypervisor 不知道该 key。
  \item CPU 还计算 integrity-check value 并存入 CPU software 不可访问的 protected DRAM。
